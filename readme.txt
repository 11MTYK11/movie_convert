※使い方

右クリックメニューから動画を変換するを選んでください

初期設定ではh.264で動画を圧縮する設定になっています

V1.0の場合
右クリックメニューにない場合は先にrun.vbsを実行してください

※アンインストール方法

V1.0以外の場合
setup.exeを実行してください

V1.0の場合
アンインストールを行う前にuninstall.vbsを実行してレジストリキーを削除してください
uninstall.vbsを実行した後にこのフォルダを削除してください
